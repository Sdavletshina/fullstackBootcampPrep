Incremental Change

Define a function incrementalChange that accepts a number and increments the global variable counter by that amount. The function should also return the current value of the counter after the increment operation. Each successive call to incrementalChange should progressively change the value held by the counter.

let counter = 0; incrementalChange(2); // => 2
incrementalChange(6); // => 8
incrementalChange(-1); // => 7
