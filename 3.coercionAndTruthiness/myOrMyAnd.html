My Or, My And

Define a function, myOr, that accepts three arguments of any type. myOr should return the same result as the buit-in || operator:

myOr(true, true, false) === (true || true || false); // => true

Then, define a function, myAnd, that accepts three arguments of any type. myAnd should return the same result as the buit-in && operator:

myAnd(1, 'apples', true) === (1 && 'apples' && true); // => true
